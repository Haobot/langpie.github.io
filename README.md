## Contribute

1. Clone langpie.github.io repository to your computer
   ```shell
   git clone https://github.com/langpie/langpie.github.io.git
   ```
2. Enter langpie.github.io folder, install dependencies
   ```shell
   npm i
   # or
   yarn install
   ```
3. Start realtime development
   ```shell
   npm run docs:dev
   # or
   yarn docs:dev
   ```
4. Build and publish
   ```shell
   npm run docs:build
   # or
   yarn docs:build
   ```
